## 🔧 脚本核心功能

### 1. 自动检测
- **事件类型字段**：自动检测包含'event'、'action'、'type'等关键词的字段
- **时间字段**：自动检测datetime/timestamp类型字段
- **事件分布**：自动分析各类事件的频率和用户覆盖

### 2. 魔法数字计算
基于4个关键行为指标：
1. **详情页查看** ≥ 5次
2. **商品点击** ≥ 8次
3. **筛选使用** ≥ 3次
4. **搜索行为** ≥ 3次

### 3. 输出报告
- **用户等级**：青铜、白银、黄金、钻石
- **达成分布**：各魔法数字的达成率
- **活跃度分析**：用户行为强度分级

## 🚨 常见问题

### Q1: 如何知道实际的事件类型名称？
A: 运行 `--explore` 模式，查看事件类型分布，根据输出调整 `event_type_mapping`

### Q2: 时间字段不是默认的怎么办？
A: 脚本会自动检测时间字段，也可以通过 `--time-field` 参数指定

### Q3: 数据量太大怎么办？
A: 使用 `--limit` 参数限制数据行数，或调整时间范围

### Q4: 需要安装哪些依赖？
A:
```bash
pip install pandas numpy pymysql openpyxl
```

### Q5: 如何自定义魔法数字阈值？
A: 直接修改脚本中的 `magic_number_config` 部分
