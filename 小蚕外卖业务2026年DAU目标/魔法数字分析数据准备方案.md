# 小蚕魔法数字分析数据准备方案

## 📊 数据准备策略选择

### 选择依据
| 数据规模 | 推荐方案 | 优势 | 注意事项 |
|----------|----------|------|----------|
| < 10万行 | Excel导出 | 操作简单，无需编程 | 文件大小可能较大 |
| 10-100万行 | CSV导出 + 脚本处理 | 兼顾易用性和性能 | 需要基础Python环境 |
| > 100万行 | 直接StarRocks查询 | 性能最佳，支持大数据量 | 需要数据库访问权限 |

### 建议流程
1. **先尝试Excel导出**：如导出失败或文件过大，转用脚本方案
2. **使用Python脚本**：自动处理数据查询、清洗、导出
3. **大数据量直接分析**：在StarRocks中直接进行初步分析

## 📋 Excel导出表头设计

### 1. 用户基本信息表头
| 字段名 | 数据类型 | 说明 | 是否必填 |
|--------|----------|------|----------|
| user_id | 字符串 | 用户唯一标识 | 是 |
| register_date | 日期 | 注册日期（YYYY-MM-DD） | 是 |
| user_type | 字符串 | 用户类型：student/whitecollar | 是 |
| city | 字符串 | 所在城市 | 是 |
| campus_company | 字符串 | 学校名称或公司名称 | 否 |
| age_group | 字符串 | 年龄段：18-22/23-30/31+ | 否 |
| gender | 字符串 | 性别：M/F/unknown | 否 |
| device_type | 字符串 | 设备类型：ios/android | 否 |

### 2. 用户行为指标表头（最近30天）
| 字段名 | 数据类型 | 说明 | 魔法数字阈值 |
|--------|----------|------|--------------|
| detail_page_views | 整数 | 详情页查看总次数 | ≥5次 |
| product_clicks | 整数 | 商品点击总次数 | ≥8次 |
| filter_uses | 整数 | 筛选功能使用次数 | ≥3次 |
| search_counts | 整数 | 搜索行为次数 | ≥3次 |
| session_count | 整数 | 会话次数 | - |
| avg_session_duration | 小数 | 平均会话时长（秒） | - |
| total_orders | 整数 | 总订单数 | - |
| successful_orders | 整数 | 成功订单数 | - |
| order_success_rate | 小数 | 订单成功率（0-1） | - |
| total_saved_amount | 小数 | 累计节省金额（元） | - |
| avg_saving_per_order | 小数 | 单均节省金额（元） | - |
| favorite_merchants | 整数 | 收藏商家数 | - |
| written_reviews | 整数 | 撰写评价数 | - |
| shared_orders | 整数 | 分享订单数 | - |
| invited_friends | 整数 | 邀请好友数 | - |

### 3. 魔法数字达成状态表头
| 字段名 | 数据类型 | 说明 | 计算逻辑 |
|--------|----------|------|----------|
| magic_number_1 | 布尔 | 详情页查看≥5次 | detail_page_views >= 5 |
| magic_number_2 | 布尔 | 商品点击≥8次 | product_clicks >= 8 |
| magic_number_3 | 布尔 | 筛选使用≥3次 | filter_uses >= 3 |
| magic_number_4 | 布尔 | 搜索行为≥3次 | search_counts >= 3 |
| magic_number_combo | 整数 | 达成魔法数字个数 | sum(魔法数字1-4) |
| magic_number_tier | 字符串 | 魔法数字等级：青铜/白银/黄金/钻石 | 基于达成个数 |

### 4. 用户留存状态表头
| 字段名 | 数据类型 | 说明 | 计算基准 |
|--------|----------|------|----------|
| day2_retained | 布尔 | 次日是否留存 | 注册后第2天活跃 |
| day7_retained | 布尔 | 7日是否留存 | 注册后第7天活跃 |
| day30_retained | 布尔 | 30日是否留存 | 注册后第30天活跃 |
| day2_retention_date | 日期 | 次日留存日期 | - |
| day7_retention_date | 日期 | 7日留存日期 | - |
| day30_retention_date | 日期 | 30日留存日期 | - |
| current_active | 布尔 | 当前是否活跃 | 最近7天有活跃 |

### 5. 时间窗口表头
| 字段名 | 数据类型 | 说明 |
|--------|----------|------|
| analysis_period_start | 日期 | 分析周期开始日期 |
| analysis_period_end | 日期 | 分析周期结束日期 |
| data_extraction_date | 日期 | 数据提取日期 |

    def estimate_data_size(self, start_date: str, end_date: str) -> Tuple[int, int]:

    def extract_user_behavior_data(self, start_date: str, end_date: str,
                                   limit: Optional[int] = None) -> pd.DataFrame:
        """
        提取用户行为数据
        这是核心查询函数，需要根据实际表结构调整
        """

        # 基础查询SQL（需要根据实际表结构调整）
        base_query = f"""
        -- 用户基本信息
        WITH user_base AS (

                u.user_id,
                u.register_date,

                COUNT(DISTINCT CASE WHEN b.event_type = 'use_filter' THEN b.event_id END) as filter_uses,
                COUNT(DISTINCT CASE WHEN b.event_type = 'search' THEN b.event_id END) as search_counts,

                -- 会话指标
                COUNT(DISTINCT b.session_id) as session_count,
                AVG(b.session_duration) as avg_session_duration,

                -- 订单指标
                COUNT(DISTINCT o.order_id) as total_orders,
                COUNT(DISTINCT CASE WHEN o.order_status = 'completed' THEN o.order_id END) as successful_orders,
                SUM(o.saved_amount) as total_saved_amount,

                -- 社交指标
                COUNT(DISTINCT CASE WHEN s.action_type = 'favorite' THEN s.merchant_id END) as favorite_merchants,
                COUNT(DISTINCT CASE WHEN s.action_type = 'write_review' THEN s.review_id END) as written_reviews,
                COUNT(DISTINCT CASE WHEN s.action_type = 'share' THEN s.share_id END) as shared_orders,
                COUNT(DISTINCT CASE WHEN s.action_type = 'invite' THEN s.invite_id END) as invited_friends

            FROM user_behavior_log b
            LEFT JOIN order_info o ON b.user_id = o.user_id
                AND o.order_date BETWEEN '{start_date}' AND '{end_date}'

            WHERE b.event_date BETWEEN '{start_date}' AND '{end_date}'
            GROUP BY b.user_id

            SELECT
                r.user_id,
                MAX(CASE WHEN r.retention_day = 2 THEN 1 ELSE 0 END) as day2_retained,

            COALESCE(ubh.detail_page_views, 0) as detail_page_views,

            COALESCE(ubh.shared_orders, 0) as shared_orders,

            COALESCE(ur.day2_retained, 0) as day2_retained,

    def calculate_magic_numbers(self, df: pd.DataFrame) -> pd.DataFrame:
        """计算魔法数字相关指标"""
        if df.empty:
            return df

        print("🧮 计算魔法数字指标...")

        # 复制数据框避免修改原数据
        result_df = df.copy()

        # 魔法数字阈值
        MAGIC_NUMBER_THRESHOLDS = {
            'detail_page_views': 5,
            'product_clicks': 8,
            'filter_uses': 3,
            'search_counts': 3
        }

        # 计算魔法数字达成状态
        result_df['magic_number_1'] = (result_df['detail_page_views'] >= MAGIC_NUMBER_THRESHOLDS['detail_page_views']).astype(int)
        result_df['magic_number_2'] = (result_df['product_clicks'] >= MAGIC_NUMBER_THRESHOLDS['product_clicks']).astype(int)
        result_df['magic_number_3'] = (result_df['filter_uses'] >= MAGIC_NUMBER_THRESHOLDS['filter_uses']).astype(int)
        result_df['magic_number_4'] = (result_df['search_counts'] >= MAGIC_NUMBER_THRESHOLDS['search_counts']).astype(int)

        # 计算达成魔法数字个数
        magic_number_cols = ['magic_number_1', 'magic_number_2', 'magic_number_3', 'magic_number_4']
        result_df['magic_number_combo'] = result_df[magic_number_cols].sum(axis=1)

        # 计算魔法数字等级
        def assign_magic_tier(combo):
            if combo == 0:
                return '未达成'
            elif combo == 1:
                return '青铜'
            elif combo == 2:
                return '白银'
            elif combo == 3:
                return '黄金'
            else:  # combo == 4
                return '钻石'

        result_df['magic_number_tier'] = result_df['magic_number_combo'].apply(assign_magic_tier)

        # 计算订单成功率
        result_df['order_success_rate'] = result_df.apply(
            lambda row: row['successful_orders'] / row['total_orders'] if row['total_orders'] > 0 else 0,
            axis=1
        )

        # 计算单均节省金额
        result_df['avg_saving_per_order'] = result_df.apply(
            lambda row: row['total_saved_amount'] / row['successful_orders'] if row['successful_orders'] > 0 else 0,
            axis=1
        )

        print(f"✅ 魔法数字计算完成!")
        print(f"   魔法数字达成情况:")
        for i, col in enumerate(magic_number_cols, 1):
            achieved = result_df[col].sum()
            total = len(result_df)
            percentage = (achieved / total * 100) if total > 0 else 0
            print(f"     - 魔法数字{i}: {achieved:,}/{total:,} ({percentage:.1f}%)")

        print(f"   魔法数字等级分布:")
        tier_counts = result_df['magic_number_tier'].value_counts()
        for tier, count in tier_counts.items():
            percentage = (count / len(result_df) * 100)
            print(f"     - {tier}: {count:,} ({percentage:.1f}%)")

        return result_df

    def export_to_csv(self, df: pd.DataFrame, filename: str) -> bool:
        """导出数据到CSV文件"""
        if df.empty:
            print("⚠️  数据为空，无法导出")
            return False

        try:
            # 确保输出目录存在
            os.makedirs('output', exist_ok=True)

            # 完整文件路径
            filepath = os.path.join('output', filename)

            # 导出到CSV
            df.to_csv(filepath, index=False, encoding='utf-8-sig')

            # 获取文件大小
            file_size = os.path.getsize(filepath) / (1024 * 1024)  # MB

            print(f"💾 数据导出完成!")
            print(f"   文件路径: {filepath}")
            print(f"   文件大小: {file_size:.2f} MB")
            print(f"   数据行数: {len(df):,}")
            print(f"   数据列数: {len(df.columns)}")

            # 显示前几行数据
            print(f"\n📋 数据预览 (前5行):")
            print(df.head().to_string())

            return True

        except Exception as e:
            print(f"❌ 数据导出失败: {e}")
            return False

    def generate_data_summary(self, df: pd.DataFrame) -> Dict:
        """生成数据摘要报告"""
        if df.empty:
            return {}

        summary = {
            'total_users': len(df),
            'date_range': {
                'start': df['analysis_period_start'].iloc[0] if 'analysis_period_start' in df.columns else '未知',
                'end': df['analysis_period_end'].iloc[0] if 'analysis_period_end' in df.columns else '未知'
            },
            'user_type_distribution': df['user_type'].value_counts().to_dict() if 'user_type' in df.columns else {},
            'city_distribution': df['city'].value_counts().head(10).to_dict() if 'city' in df.columns else {},
            'retention_rates': {
                'day2': df['day2_retained'].mean() if 'day2_retained' in df.columns else 0,
                'day7': df['day7_retained'].mean() if 'day7_retained' in df.columns else 0,
                'day30': df['day30_retained'].mean() if 'day30_retained' in df.columns else 0
            },
            'behavior_stats': {
                'avg_detail_views': df['detail_page_views'].mean() if 'detail_page_views' in df.columns else 0,
                'avg_product_clicks': df['product_clicks'].mean() if 'product_clicks' in df.columns else 0,
                'avg_filter_uses': df['filter_uses'].mean() if 'filter_uses' in df.columns else 0,
                'avg_search_counts': df['search_counts'].mean() if 'search_counts' in df.columns else 0
            },
            'magic_number_stats': {
                'avg_magic_combo': df['magic_number_combo'].mean() if 'magic_number_combo' in df.columns else 0,
                'tier_distribution': df['magic_number_tier'].value_counts().to_dict() if 'magic_number_tier' in df.columns else {}
            }
        }

        return summary

    def run_extraction(self, start_date: str, end_date: str,
                       limit: Optional[int] = None,
                       export_filename: str = None) -> pd.DataFrame:
        """运行完整的数据提取流程"""

        # 连接数据库
        if not self.connect():
            return pd.DataFrame()

        try:
            # 1. 估算数据量
            user_count, record_count = self.estimate_data_size(start_date, end_date)

            # 2. 获取表结构（可选）
            # schema = self.get_table_schema()

            # 3. 提取数据
            df = self.extract_user_behavior_data(start_date, end_date, limit)

            if df.empty:
                print("⚠️  未提取到数据，请检查查询条件")
                return pd.DataFrame()

            # 4. 计算魔法数字指标
            df = self.calculate_magic_numbers(df)

            # 5. 导出数据
            if export_filename:
                if not export_filename.endswith('.csv'):
                    export_filename += '.csv'
                self.export_to_csv(df, export_filename)

            # 6. 生成数据摘要
            summary = self.generate_data_summary(df)

            print("\n" + "="*50)
            print("📈 数据提取摘要报告")
            print("="*50)
            print(f"用户总数: {summary.get('total_users', 0):,}")
            print(f"时间范围: {summary.get('date_range', {}).get('start')} 到 {summary.get('date_range', {}).get('end')}")

            if 'user_type_distribution' in summary:
                print("\n用户类型分布:")
                for user_type, count in summary['user_type_distribution'].items():
                    percentage = (count / summary['total_users'] * 100) if summary['total_users'] > 0 else 0
                    print(f"  - {user_type}: {count:,} ({percentage:.1f}%)")

            if 'retention_rates' in summary:
                print("\n留存率:")
                for period, rate in summary['retention_rates'].items():
                    print(f"  - {period}: {rate:.2%}")

            if 'magic_number_stats' in summary:
                print(f"\n魔法数字达成情况:")
                print(f"  - 平均达成个数: {summary['magic_number_stats'].get('avg_magic_combo', 0):.2f}")
                if 'tier_distribution' in summary['magic_number_stats']:
                    for tier, count in summary['magic_number_stats']['tier_distribution'].items():
                        percentage = (count / summary['total_users'] * 100) if summary['total_users'] > 0 else 0
                        print(f"  - {tier}: {count:,} ({percentage:.1f}%)")

            return df

        finally:
            # 确保关闭数据库连接
            self.disconnect()
