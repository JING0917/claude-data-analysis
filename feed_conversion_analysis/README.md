# 小蚕首页feed流转化率分析 - 晓晓 vs 小蚕

## 项目背景
分析首页feed流中晓晓商品卡片转化率比小蚕高1个百分点的原因，优先验证假设A（流量质量差异）。

## 分析框架

### 假设验证优先级
1. **假设A：流量质量差异** - 晓晓吸引的用户质量更高（更易转化）
2. **假设B：商品属性差异** - 晓晓商品本身更具吸引力  
3. **假设C：展示位置效应** - 晓晓卡片在feed流中的位置更优
4. **假设D：用户群体差异** - 点击晓晓和小蚕的用户画像不同
5. **假设E：数据统计偏差** - 已确认口径一致，暂不考虑

### 分析维度（按优先级）
1. **平台/版本分布分析**（分析4）- 不同平台(iOS/Android/H5)的转化率差异
2. **用户重合度分析**（分析1）- 同时点击晓晓和小蚕的用户行为对比
3. **地域分布分析** - 区县级别的转化率差异
4. **行为漏斗分析** - 点击→详情页→报名的转化路径对比
5. **趋势分析** - 最近30天每日转化率变化

## 数据来源
- **主表**: `dws.dws_sr_traffic_homepage_mix_ascribe_d`
- **时间范围**: 最近30天（按日统计）
- **数据粒度**: 日期 + 区县 + 平台 + 版本 + 活动类型
- **关键字段**:
  - `activity_type`: '小蚕活动' 或 '晓晓'
  - `clc_num`: 点击量
  - `baoming_order_num`: 小蚕报名订单量  
  - `xx_baoming_order_num`: 晓晓报名订单量
  - `detailpage_pv`: 详情页PV

## 分析流程

### 第一阶段：数据提取与清洗（1天）
1. 执行SQL查询，提取30天数据
2. 数据质量检查与清洗
3. 导出为CSV/Excel格式

### 第二阶段：核心分析（2天）
1. **平台/版本分布分析** - 对比不同平台的转化率差异
2. **用户重合度分析** - 分析用户点击行为模式
3. **地域分布分析** - 识别高转化区县

### 第三阶段：深度分析与报告（2天）
1. 行为漏斗对比分析
2. 趋势分析与异常检测
3. 生成钉钉文档格式报告
4. 提出业务优化建议
