# Amazon饰品项目失败复盘

## 结论

**项目在确认数据范围内净亏约$13,822（¥99,518），加上未计入的库存积压¥130,000，实际亏损远超30万RMB。核心死因：广告投放效率极低（整体ACOS 120.7%），几乎一半广告费（46.1%）花在ACOS>150%或零销售的无效活动上。产品层面毛利率66.6%是健康的，但广告费+平台费合计吃掉收入的111%，产品毛利根本兜不住。**

---

## 口径说明

| 指标 | 口径 |
|------|------|
| 收入 | 业务报告「已订购商品销售额」(24/7→26/5) + 订单明细4月销售额 |
| 广告费 | 广告活动报告「总成本」汇总（50个活动，全部已暂停） |
| 货品成本 | 用户提供：$4/SKU 统一采购价 |
| 平台佣金 | 按Jewelry类目17%估算（实际按ASIN售价阶梯可能略有差异） |
| FBA配送费 | 按小标准件$3.22/件估算（2024年费率） |
| 数据缺失 | 2023.7-2024.3（约10个月）、2024.5-2024.6（2个月）数据不可用；仓储费、弃置费未计入；商品退货数据未计入 |

## 数据分析

### 1. 收入与销量趋势

业务报告覆盖23个月（24年7月→26年5月），加4月订单明细，可确认的总销售额$31,733、2,589件。

| 阶段 | 月均销售额 | 趋势 |
|------|-----------|------|
| 2024.4（订单明细） | $10,142 | 启动期（单月最高） |
| 2024.7-2024.12 | $2,962 | 稳定期 |
| 2025.1-2025.6 | $532 | 断崖下跌 |
| 2025.7-2026.5 | $32 | 几乎归零 |

销售额在6个月内从$4,972跌至$717，下降85.6%，此后未恢复。

### 2. 成本拆解

以可确认数据（$31,733收入）为分母：

| 费用项 | 金额 | 占收入比 |
|--------|------|----------|
| 货品成本 | $10,356 | 32.6% |
| 广告费 | $21,468 | **67.7%** |
| 平台佣金(估) | $5,395 | 17.0% |
| FBA配送(估) | $8,337 | 26.3% |
| **合计** | **$45,555** | **143.6%** |

每卖一件均价$12.26的商品，成本结构如下：
- 货品：$4.00
- 广告：**$8.29**（每件广告费）
- Amazon收费：$5.30
- **每件净亏：$5.33**

### 3. 广告效率——致命伤

50个广告活动，全部SP/SD类型，全部已暂停：

| 效率分层 | 花费 | 占比 |
|----------|------|------|
| 有效（ACOS<100%） | $5,768 | 26.9% |
| 临界（ACOS 100-150%） | $5,807 | 27.0% |
| **纯亏（ACOS>150%或零销售）** | **$9,893** | **46.1%** |

**亏损三巨头**（花费$8,221，占总量38.3%）：

| 活动名称 | 花费 | ACOS | 
|----------|------|------|
| gold earrings for women-精准-5.9 | $4,006 | 171% |
| gold earrings for women --精准-3.14 | $2,147 | 168% |
| earrings-精准-2 | $2,068 | 165% |

这三个活动的平均CPC高达$1.74-$2.35，而产品售价仅$12.99。每件广告成本$8-10，产品毛利$8左右——**广告本身就亏，产品毛利填不上**。

### 4. 少数有效的活动

仅2个活动ACOS<70%：
- earring set-词组：花费$899，ROAS 1.54
- gold plated earrings for women-词组：花费$83，ROAS 1.54

这些是词组匹配+低CPC的组合，说明精准长尾词有机会跑正。

## 验证

### 可确认的部分
- 广告报告50个活动的花费$21,468与收入$31,733存在交叉（广告驱动销售占比约56%），实际有机销售约44%
- 按ACOS 120.7%的全局广告效率，即使只看广告驱动的销售（$17,789），广告费$21,468已经超出$3,679——**广告本身是负ROI的**
- 产品毛利率66.6%不足以覆盖67.7%的广告费率+43.3%的平台费率

### 不能确认的部分
- 缺失的早期数据（23.7-24.3）中可能包含更高或更低的广告投入
- 实际FBA费用（仓储费、长期仓储费、弃置费）未计入，实际亏损更大
- 退货/退款造成的额外损失未计入
- $4采购成本为用户估算，如实际更高则亏损更大
- 家中积压¥130,000库存为已支付但未售出的货品成本，属于额外沉没成本

### 总体判断
**即使补齐所有缺失数据，结论方向不会改变**——这是一个广告投放效率极低、单位经济模型（Unit Economics）为负的项目。核心问题不在选品（66.6%产品毛利在饰品品类不算差），在广告投放策略：高CPC精准匹配词（$1.74-$2.35/click）打低价产品（$12.99），数学上无法跑正。
